What is React?

React is a JavaScript library for building user interfaces. It allows developers to create large web applications that can change data without reloading the page. Its key feature is the ability to build components that manage their own state, making it easier to develop interactive UIs.

Creating and nesting components

React apps are made out of components. A component is a piece of the UI (user interface) that has its own logic and appearance. A component can be as small as a button, or as large as an entire page.

React components are JavaScript functions that return markup:


            function MyButton() {
            return (
            <button>I'm a button</button>
            );
          }
